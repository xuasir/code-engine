# Vona JS 生成器目标设计说明

* Vona JS 的运行时注入需求主要是为了支持Web前端应用运行所必须的资源注入，包括但不限于：

  - Vue 的初始化流程
  - Vona 运行时插件的加载解析
  - Vona 层资源的注入
  - Vona 能力的类型安全注入

* Vona JS 生成器工作在 .vona 目录下，生成的文件包括但不限于：

  - 类型文件（`.d.ts`）
  - Vue/React文件（`.vue`/`.tsx`）
  - TS文件（`.ts`）
  - JSON文件（`.json`）

* Vona JS 生成器的文件写入策略：

  - 静态的文件写入：直接复制到目标目录
  - 动态的文件写入：根据 Vona 层资源的合并结果生成
  - module协作文件写入： 多个 Vona module 之间的协作构建一个文件，最终合并写入

* Vona JS 生成器的文件生成策略：

  - 内存模式：所有文件生成在内存中
  - 磁盘模式：每个文件生成后立即写入目标目录
