# @vona-js/plugin-vue

Vona 与 Vue 的集成插件（开发中）。

## 安装

```bash
pnpm add @vona-js/plugin-vue
```

## 使用

当前导出：

```ts
import { one, two } from '@vona-js/plugin-vue'
```

更多能力（如 Vue 插件与组件自动接入）将逐步完善。

## 开发与测试

- 构建：`pnpm build`
- 测试：在仓库根目录执行 `pnpm test`
- 类型检查：`pnpm typecheck`
- Lint：`pnpm lint`

## 构建与发布

- 使用 tsdown 构建：`pnpm build`
- 工作区统一发布：在仓库根目录执行 `pnpm release`
